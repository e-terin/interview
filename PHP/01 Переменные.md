#### Как передаются в параметры функции переменные? Объекты?
Переменные по значению, объекты по ссылке.

#### Как передать переменную по ссылке?
Через амперсант:
```php
$a = 1;

// Внутри функции меняем значение переменной
function increase(int &$value){
    $value++;
}

increase($a);
var_dump($a); 

```
